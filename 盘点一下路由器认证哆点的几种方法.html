盘点一下路由器认证哆点的几种方法

发表于 2022-04-17 更新于 2022-04-19 分类于 校园网折腾 Waline： 阅读次数： 本文字数： 762 阅读时长 ≈ 1 分钟

全部基于GitHub项目：drcoms/drcom-generic: Dr.COM/DrCOM 现已覆盖 d p x三版。 (github.com)](https://github.com/drcoms/drcom-generic)

只是项目有不同的实现模式

所有的前提都是wireshark抓包获取config文件

1 python认证

特点：路由器要有python

  1. 获得学校对应的py程序drcom-generic/custom at master · drcoms/drcom-generic (github.com)

  2. 将程序中对应的用户信息更换为自己的信息

  3. 将drcom.py重命名为drcom，将其放置在路由器/usr/bin/地址下（防止开机删除）

  4. putty直接python /usr/bin/drcom

  5. 成功效果是一篇空白

2 dogcom二进制文件认证

特点：不需要python环境，直接就能用

  1. 去恩山论坛下载对应固件的dogcom二进制文件

  2. 将dogcom和drcom.conf上传到/usr

  3. dogcom-右键-属性，将八进制表示更改为0777，赋予dogcom权限

  4. putty直接/usr/dogcom -m dhcp -c /usr/drcom.conf -v即可

输入netstat -tunlp可查看目前的端口使用情况

可输入kill -9 pidnum关闭相关进程

3 dogcom.ipk认证

特点：直接ipk安装到路由器认证，独立方便，性能要求高一点

  1. 恩山论坛下载x86的ipk文件
  2. 将dogcom.ipk and drcom.conf两个文件上传到/usr/目录下
  3. putty opkg install /usr/dogcom.ipk安装ipk
  4. putty dogcom -m dhcp -c /usr/drcom.conf使用该命令配置dogcom.ipk
