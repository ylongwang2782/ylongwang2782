openwrt刷入dogcom认证哆点NOTE

发表于 2022-04-09 更新于 2022-04-19 分类于 校园网折腾 Waline： 阅读次数： 本文字数： 1.6k 阅读时长 ≈ 1 分钟

登录信息抓包

需要的软件为

1.wireshark

  1. 将网线连接电脑
  2. 打开客户端，输入账号和密码
  3. 打开wireshark，点击以太网，点击start
  4. 登录哆点
  5. 大概十几秒后，点击左上角红色按钮停止
  6. File-Save As 保存为dr.pcapng

2.生成配置文件

  1. 登录在线配置生成器https://drcoms.github.io/drcom-generic/
  2. 选择对应的哆点版本
  3. 将之前获得的dr.pcapng上传
  4. 获得新的配置文件，点击SAVE保存配置文件drcom.conf
  5. 用记事本打开配置文件drcom.conf，再对应的password处填入密码并保存

脚本上传路由器

  1. 打开winscp软件，地址输入路由器管理地址，账号，密码
  2. 在winscp界面中，进入到/usr地址
  3. 上传dogcom和drcom.conf
  4. dogcom-右键-属性，将八进制表示更改为0777，赋予dogcom权限

运行脚本

  1. 将网线接入到路由器的wan口
  2. 使用putty登录路由器
  3. /usr/dogcom -m dhcp -c /usr/drcom.conf -v

添加开机自启

在WinSCP下打开/etc/rc.local

在exit 0上一行输入下列命令：

/usr/dogcom -m pppoe -c /usr/drcom.conf
