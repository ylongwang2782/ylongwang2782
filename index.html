PVE黑群晖直通硬盘

发表于 2022-03-28 Waline： 阅读次数： 本文字数： 770 阅读时长 ≈ 1 分钟

目前的b站还不支持markdown语言，所以不准备写很多

写这些的目的主要是把自己以后可能用得上的东西自己重新整理一下，免得找不着了，很多收藏的东西，总是找不到，还是得自己总结的东西逻辑清楚

黑群晖因为特殊性，加装硬盘需要特殊方法。

本文仅用于PVE下安装的群晖，不过估计其他的虚拟机装的群晖也能适用。

首先准备一块硬盘

PDD买的一块海康威视的监控盘，支持7x24就行，其他的便宜就行，反正只准备用他跑PT，烂了就烂了。重要的资料还是用PC的机械硬盘单独拷贝。

安装相关环境

在PVE的SHELL里输入以下指令

1、apt-get update 更新环境

2、apt-get install lshw 安装lshw

3、ls -l /dev/disk/by-id/ 查看已插入硬盘信息

得到一大串神秘代码

只看ata开头的硬盘ID

可以看出，我插入的四块磁盘，每个型号都能简单的辨识

第一块是我的启动盘，也就是安装的PVE和NAS只有16G的一个msata

第二块是我插入一快垃圾500G硬盘

第三块是我之前插入的一块西数WDC4T紫盘，

第四块就是我最新的一块盘，可以看出和前面的那块紫盘基本相似。

选取不带part的那行语句

把他单独拎出来

然后根据下面我的格式填充即可

qm set 101 –sata4 /dev/disk/by-id/ata-WDC_WD40PURX-78AKYY0_WD-WX32DA1LTP3S

其中，101为NAS在PVE中的标号

sata4则为NAS中第几块硬盘，从01234挨着排下去就行，别填错了就ok

结束，此时硬盘已经直通到nas里了

后面在群晖里只要设置一下就行了

NAS里设置

在存储空间管理员里，新增

选择灵活性提高

raid类型选择basic，PT盘差不多得了

把要用的硬盘拖过来下一步就完成

2021年春，我卖掉了iPad，买了MacBook

发表于 2021-06-23 更新于 2022-03-28 Waline： 阅读次数： 本文字数： 2.8k 阅读时长 ≈ 3 分钟

卖掉了原来的iPad air3

写在前面：ipad air3是2019年暑假买的，2021年春卖了。

为什么呢？

东西不好用，一直吃灰，在图书馆的日子，总觉得ipad不顺手，很多时候也就看看网课，pdf。更多的时候是放在书架上凑数。很多时候的功能好像用手机也行？虽然手机没有平板大，但是

  • 手机便携，随时带在身上，要看ppt，pdf的时候只要马上掏出来就可以了。
  • 手机自带网络。校园网并没有家里的Wi-Fi那么好使，很多地方覆盖不了，并且认证麻烦，除非提前把文件下到平板里，不然也没有手机方便。
  • 手机打字方便。随便遇到什么文档马上修改，或者有什么idea要马上记录，手机可以马上记录，同是触摸输入，手机要比平板顺手多了。如果想要超过手机，平板得配上一个超重的键盘。

综上，平板只有在固定位置，长时间不移动，作为一个固定工具使用，这样才会比手机好使

嗯？平板的定义不是便携式电脑，更轻便版的笔记本电脑吗？？

哦？但是我丝毫没有感受到我使用的是一个笔记本电脑！大概只是一个带tvOS的显示器吧

于是

卖了

2019年6月买成3300，21年4月卖成2600.一年半多，交了600的租金

所以，我更需要的是一台真正的笔记本电脑来帮助我的工作，用手机辅助电脑。

而不是出门在外用平板辅助手机。

当然，我也把我的游戏本也卖了

买了MacBook pro

为什么买macOS？

思来想去，想着既然是要换电脑，无非就是macOS和win两种。

我选择了MacBook

  1. 我的需求。能够看网课，看pdf，看ppt，并且能够方便记笔记（指有键盘，个人认为pencil并不能很高效记录笔记）
  2. 寝室里已经有一台win系的台式电脑，重型任务，以及win的工作可以交给台式来做。可以用作mac没有适配的一些软件
  3. win系的二手市场看得我眼花缭乱，各种配置各种群魔乱舞，反观mac的市场就比较简单了，仅仅是MacBook12寸，mba和mbp的选择而已，每年的机型，加上每年的配置（当然是最低配）
  4. 生态绑定。向来是喜欢iOS的设计，所以一直是使用的iPhone，并且购买的watch作为辅助工具。如果平时在外，要用电脑为主手机为辅，这个时候选择同一个生态系统的硬件应该会更加方便。后来的事实证明我是对的。

选择哪一款MacBook？

虽然mac的系列比较少，但是也需要做一些的功课

首先第一步就是了解各大机型和配置。

我选择了Microsoft edge浏览器中的集锦功能作为我的工具。

Microsoft edge的集锦功能

借助新的 Microsoft Edge 浏览器中的”集锦“功能，可以帮助你记录和跟踪浏览网页时的想法。无论是购物，计划旅行，收集研究笔记或课程计划的笔记，还是只是想在上次浏览的地方继续浏览，集锦功能都能提供帮助。

集锦会在你登录的设备之间同步，因此，如果你使用多台设备，则所有集锦上的收藏都将始终是最新的。

引用自官方介绍

集锦支持插入文本，插入URL，插入截图

集锦是一个强大的浏览器信息收集帮手。

我在集锦中收集三种MacBook的参数配置，并加以自己的文字总结，以挑选最适合自己的MacBook。

这样可以帮助我迅速找准目标

我找出来了以下要点

  1. 屏幕。总体分为视网膜屏和非视网膜屏，也就是Retina屏，也称R屏。视网膜屏意味着更好的分辨率，更好的ppi，更好的色彩，更好的亮度。
  2. 屏幕边框。较宽的变宽，例如早期的mab系列和MacBook12寸系列。
  3. 键盘虽然有蝶式和剪刀，不过对于我来说则是无所谓。
  4. 处理器。至少得满足Office和视频播放功能
  5. 硬盘。对于我的需求，128G足矣
  6. 内存。8G足矣

作为颜值党，R屏的选择，让我几乎排除了早期的MacBook，以及加上性能上的要求。

我的目标主要定在17年的MacBook12寸、18年以后的mba、15年后的mbp。

锁定目标后，下面的功夫就全部在闲鱼了。

毕竟东西再合适，得先符合荷包。

上闲鱼

本着并没想在闲鱼上捡漏，只是找一个信得过合适的卖家就行。

于是蹲了大概四五天，遇见一个4000元的mbp17丐版。东西是19年3月购于京东，使用时间较短。

为什么排除了MacBook和mba？答：闲鱼上都是卖传家宝的。

开箱

卖家连原包装都保留着，让我还能有机会体验到一点原开箱的感觉

东西还是不错的，17年丐版的mbp配置依然超群。

13.3寸的R屏，搭配2560*1600的分辨率，ppi高达227，同时亮度支持到500nit！也支持P3色域。

8G+128G的内存和硬盘配置也是完全够用。

当然，捡二手，必然还是会有一些成色问题。

我这一天就是两处外观问题

\1. command键掉漆

\2. 右下角有磕碰

最后

东西到手已经使用了三周了。目前感觉良好，心里还有点小赚

从实际体验上来说，这次的接盘是完全正确的。因为自从有了mbp后，我的大部分的文字操作基本都可以在mbp上完成了，日常的刷张大妈，刷刷b站，看看qq完全都放在了mbp上完成。

  • mbp休眠使用的逻辑更像一台手机，因为你基本不会将它关机，不用时只需要轻轻合盖就可以了。使得mbp的使用门槛降低很多，不会再有像我去打开台式一样有那么的流程，那么多的思想负担。
  • 同时，mbp的软件，特别是浏览器，则是电脑版的使用逻辑，这使得我用起来得心应手，丝毫没有感觉到很不方便。这一点大概也是和平板最大的区别吧

MacBook和平板差距大吗？

很大，MacBook才是生产力的代表

不，他没有键盘

平板不是可以用来打游戏吗？

抱歉，我并不玩手游

不，手写记笔记的速度完全比不过键盘。况且为什么我不用纸质的记笔记呢？

哈哈，笑话

不，我的专业并不需要画画

平板不是有超强的续航吗？

上面的灵魂发问仅仅是我的个人看法。

我的真实体验是

手机+笔记本电脑是刚需套餐

ipad仅仅是一个大号iPhone，ipad能做的iPhone都能做，MacBook能做的iPad都做不了

iPad能提供的，是一个更大的屏幕，你可以用它在大屏上畅玩手游，愉快地刷爱奇艺。

2021年春我在闲鱼3700淘了一个iPhone11

发表于 2021-06-23 更新于 2022-03-28 Waline： 阅读次数： 本文字数： 3.5k 阅读时长 ≈ 3 分钟

在换手机之前手里是18年上大学买的一个iPhone8红色专版，这个版本当初在重庆北城天街那个旗舰店买成5888，这个价格一直心疼了我很久，毕竟现在它在闲鱼就只能卖1000出头，以至于我心痛到不想把它卖了。当然也要排除我这两年的裸奔导致的伊拉克成色导致他这么不好卖。

说说这个红色专版的iPhone8吧。这个当初买的时候就被告知，每当apple卖出一个iPhone red product ，就会相应的捐助非洲难民。当然我不是因为他是红十字的红色，而是因为中国红而买它的。本来在去店的轻轨上就想好了要买一个红色，但是当去了实体店还是犹豫了一下，心想红色会不会太骚了，没想到旁边的亲妈看到了，直接就说：买黑色干嘛啊？要来就来个亮色的，来点挑战。这一下就坚定了我买红色的决心。最开始在大一的时候还有点不好意思在公众场合把我红色的手机拿出来，毕竟那个时候基本大家的手机都是黑白，彩色的一般都默认是女生使用的，哈哈哈。后来逐渐脸皮厚了，也就放开了，这就是我的中国红！

不过iPhone8在我手里不知道是什么原因，经常两三个软件杀后台，可能是它的A11芯片太弱了吧。其实杀后台和打开软件慢了一点这两个缺点我本来还是能够忍受的，但是随着我开始对拍照感兴趣就发现了这个小8最大的问题！打开相机十分缓慢！！每次打开相机大概需要两秒左右的黑屏吧！更难受的是，如果是在微信或者qq里调用拍照几乎要将手机卡死，因为这个原因我被真正卡死2分钟几次了，就是那种卡住按什么键都没反应，锁屏也没用。这让我忍无可忍，只能思量着换一个手机了。

选哪一款机型呢？

不过换一个手机又岂是那么简单的，思量了半天，最后决定iPhone11最为目标，虽然已经是2021年春，秋季发布会马上就要来了。但是对比iPhone12和iPhone11就会发现， 12对比11的升级方面，仅仅是多了一个5g，处理器升级，以及hdr的显示和拍摄，Dolby vision的支持，并同时将电池给砍了30%。总体来说就是多了5G，录像提升了，电池减小了，屏幕变oled了。结合我个人的使用的目标就不难发现，首先我没有5G卡，我主要是拍照多，屏幕无所谓，以及囊中羞涩。所以买11就更为合理。

至于比11更低的版本，那还是算了，可能不大适合我的需求，毕竟我手速太快，换一个提升不大的机型其实不划算，手机毕竟还是作为工具的一种，总不能太垃圾了吧。

一手还是二手呢？

那么，到底是买全新还是二手呢？经过上次买apple watch的经历我已经完全沉迷于淘二手的欢喜之中了。

于是这次我毫不犹豫的选择了去海鲜市场淘二手，毕竟去年双11，11才3999，这让我买一手的心也凉了半截。二手市场的价格还是比较稳定的。

买二手，买到好的东西是一回事，其实淘二手这个过程也是十分快乐的，有那种自己赚了很多的喜悦，和一种赌的兴奋。毕竟二手东西不像一手那样一尘不变。东西可能成色完美，可能配件齐全，可能盒子发票全在，可能还有一堆赠送，可能还有一个说不完的故事。二手的东西太奇妙了。

目前大概行情

开淘

去酷安看了一下，手机基本都是各种黑解各种卡贴等等，什么例如美区无锁等词语。这些内容对于我仅剩的摸鱼时间，我可能没有这么多的时间和精力成本去搞定这其中的种种奥妙，所以我直接排除了各种非官机。

在闲鱼淘二手，没有关键词是不行的。怎么将非官排除在外呢？这个其实很简单，只需要在iPhone11后面加上京东的关键词就可以。

加上京东的关键词，搜索内容一下就清楚很多了，基本将各种牛鬼蛇神给排开了，剩下的选项基本都是个人卖家出闲置，并且在京东还有购买记录和发票，大部分也都还有盒子等。

毕竟这一款iPhone11我还没买就已经准备它的二手出路了。

想要将二手价格卖的好，卖的快，东西成色一定要好，盒子发票一定要有，购买渠道一定要明白。

秉承这几点的思想，去淘一个二手的京东货或许是将来卖二手的一个很好的保障。

直接闲鱼搜索：iPhone11 京东，先总体浏览一下价格，价格的参考变量大概有：

机型，内存，成色，箱子，电池健康，充电器，盒子，发票，以及卖家一些配件，例如手机壳之类的。

机型是决定价格的绝对参数，直接将11分为三档，普通版最便宜，pro其次，pro max最贵

内存：第二个决定参数，128大致要比64多值200-300不等。而且由于64的需求量较少，更多人选择128G，所以64G基本上是相当便宜的。

成色：完美成色带壳子贴膜，有轻微刮痕，有一两处磕碰，有很多划痕和深划痕到伊拉克战损版本，价格也是随之降低的。

电池健康：100%，95%，90%等，也相当影响价格。

实际使用体验

屏幕

对比iPhone8，最为直观的就是屏幕。屏幕由非全面屏的4.7到全面屏的6.1，整整大了30%，而且因为是对角线尺寸，实际应该在40%左右的提升！！没有了下面的home键，视觉的注意力更加集中到屏幕的显示内容。虽然有大黑边有大刘海，不过还是体验相当好。

至于分辨率和颜色，倒是没有什么感觉。

电池

使用了几乎三年的iPhone8的电池早已经不行，虽然显示电池健康80%，但实际体验已经十分难受，仅仅知识qq和music的使用就导致了我每天三冲。基本上有电就冲已经成为了我的习惯。

更换到iPhone11后，电池提升到了3100，相比于iPhone8的1800，整整提升了72%，实际使用体验就更为明显了。每天早上充满电，晚上11点到寝室基本上还能剩下30的电量，真的太棒了！

处理器

可以明显感受到处理器带来的提升，打开qq从2s时长到0.5s左右。特别是相机的打开，配合面部识别，只需要将手机一拿起来然后向右一滑就可以达到秒拍摄的感觉。相机打开速度基本在0.3s左右，相当快。这使得我最近每天基本都是五张照片起步，真的拍照无压力的啊，看到想拍的，举手拍就是了，这种无任何思想负担的优化才是真正我想要的使用体验。

除了日常使用软件的开启速度变快，多任务同时进行也完全没有问题了。基本上同时开四五个软件都不会杀后台，虽然我已经养成了手动杀后台的习惯，后台基本不会超过三个。

Face ID

由触控提升到faceid，我的主观上是更舒服了。

faceid，再也不会因为手沾水而解锁不了。也不需要再去用大拇指去摸索那个home键了。也不需要有通知的时候还要先点通知再指纹解锁

faceid配合拿起唤醒，让我轻松掏出手机就能使用，又是减少使用障碍的一点。

虽然face id也有些不好，比如我脱了眼镜就不认识我了，尴尬。

重量和大小

这个不得不说一下，真的还是重了不少，不多不少，基本上也是重了30%和屏幕的提升接近，对于小手人士和追求轻便的人来说，这30%真不好说

最后

手机终究是一个工具，虽然越来越多的人在手机上刷抖音看视频玩游戏，但是回到工具这一点来说，iPhone11对于我来说，它是一个合格的工具，满足我的通讯需求，满足了我记录生活的需求。至少不像iPhone8那样卡得让我不想掏出来，哈哈哈

2021年，apple watch s3还有一战之力吗？

发表于 2021-03-28 更新于 2022-03-28 Waline： 阅读次数： 本文字数： 2.9k 阅读时长 ≈ 3 分钟

写在前面

为什么到了2021年我买了一块apple watch？又为什么是apple watch s3？

首先就是作为使用了iPhone8三年的用户，我还会继续坚持ios系统手机的。继续坚持ios生态。

原因如下

  1. 虽然现在小8变得日常卡顿，两三个app就开始杀后台，但是对于我也远远够了，日常的qq微信加音乐还有zdm使用还是挺流畅的。证明一台iPhone手机对于我来说，寿命基本大于等于3年。在这个使用寿命的基础上，入手ios生态圈真的不亏。
  2. 另外，在软件和系统上，我还特别喜欢apple的设计，真的是超级喜欢，简洁，线性，丝滑，干净。虽然不能有更多的骚操作，但手机对于我来说就是单纯的获得信息的一个工具。不玩手游，性能就没有那么吃紧，一个更简单的单纯的手机就更适合我。
  3. 还有一个特别的原因，线性马达，用过iPhone的人就知道，apple家的这个线性马达是真的上瘾，曾经一直以为我的home键是可以按下去的，直到我关机了再尝试了一下才发现， 原来home键是硬的！按动的感觉全靠震动模拟的！太神奇了。
  4. 苹果家的东西配件是真的好买啊。从电池，屏幕到主板背板，基本上都能很便宜的更换！这一点我相信其他手机一定做不到iPhone这一点。秉承着能修就修，能用就用的原理，一个广阔的配件市场是能增加我的使用体验的。
  5. 及其活跃的二手市场。iphone、mac、ipad、airpods、watch在二手市场的流通太广了，数量的提升带来了更多的捡漏机会，嘿嘿。不论是在学校跳蚤市场，还是闲鱼上，苹果产品都多如牛毛，经常蹲一蹲，捡捡漏。并且就算不用了，出二手也十分快捷。不得不说apple pencil真的太保值了，19年暑假600买的一根pencil，用了一年550卖给了别人。相当于花50租了一年，几乎等于零成本体验pencil。就是基于此特点，一块昂贵的4位数apple watch，经常能以三位数体验。

综合以上原因，我购买了 苹果手表而不是其他家的智能手表。

至于为什么又是s3呢？

  • 功能要求。作为三种仅售的watch，s3被苹果官方定义为低端体验机。其具备的锻炼跟踪，游泳防水，手表通话，睡眠，心率，地图基本涵盖一个apple watch最基本的功能。对于一个最初入手watch的我来说，是一个低成本体验watch的最合适选择。
  • 价格要求。虽然全新的s3最低也要千位数起步，但是因其广泛的更替和闲置原因，导致在闲鱼上有很多的交易机会。经过长时间的观察最终确定s3的价格基本在500-1000。非常符合我的预算。并且就算是不适合，二手买来再三手卖出去，损失也不会很大。因为市场很大，所以不用考虑二手交易会消耗很大的精力。同样不考虑一手的原因包括一手到二手的折价太大了！学生党承受不起。

基于以上两点要求，s3就是最适合我的一款watch。

购买过程

既然是准备在闲鱼上购买了，那么这个购买过程一定还是要记录一下的。

以前就有过闲鱼买卖的经历，有赚有亏。不得不说，在闲鱼上冲浪，真的看一个胆子大，手速快。

为了这次交易，我特别学习到了一些小技巧。

  1. 时间。发布时间越早越好，只看1天内或者7天内的。发布时间越久的，不是价格太高就是东西有问题，买了不用说肯定是你亏。

  2. 购买方。不要购买闲鱼商家的，不要购买闲鱼商家的。既然是捡漏了，肯定是想要在二手交易过程中赚到，都不想亏到。如果去买商家的，不用说，肯定是他赚你亏，不论东西多便宜还是照片多精致，他绝对不会亏，不然他还搞什么生意？所以，想要赚到，就只买个人卖家的。区别方法，看动态和上架东西数目。几百上千的直接pass。

  3. 信誉。信誉仅作为参考，只能说高的可能好，低的也不一定坏。最重要的分辨方法，就是分析卖家的动态，分析他为什么要卖？分析他的心理价格。

  4. 评定标准。这个最重要，为什么同样一个东西，有的人卖得很贵？有的卖得很便宜？这其中关乎着一个时间。卖家越急他就卖得越便宜，买家越急，就买得越贵。所以，收二手一定不要急。记住，只要你肯等，一定是你赚。耐心是唯一的优点。

就像上面那个例子，这位老哥一直想让我买他的。在这个过程，明显就是他着急了。谈到这里，我大概就知道了，买他的肯定是血亏，所以一直使用缓兵之计。
