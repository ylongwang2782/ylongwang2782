群晖怎么访问GitHub

发表于 2022-06-23 Waline： 阅读次数： 本文字数： 326 阅读时长 ≈ 1 分钟

背景

因为安装hacs需要GitHub认证，但是家里的群晖无法访问GitHub，所以需要采取特殊手段。

方法设计

  • 群晖安装clash（太复杂）
  • 群晖使用pc的clash

综上，选择第二种方法。

实现方法

clash拥有 allow lan的功能，即局域网使用功能

  1. 在clash中开启allow lan
  2. cmd 中 输入ipconfig 查询本机ipv4地址，或者其他方法都可以。
  3. 群晖 - 控制面板 - 网络 - 代理服务器 - 地址填写pc的ip地址；端口填写默认的clash 端口 7890（或者修改后的端口）
  4. 勾选通过代理服务器连接，同时pc的clash开启proxy

以上，群晖就可以正常访问GitHub了
