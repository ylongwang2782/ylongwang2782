X86安装openwrt

发表于 2022-04-10 更新于 2022-04-19 分类于 路由器 Waline： 阅读次数： 本文字数： 1.3k 阅读时长 ≈ 1 分钟

准备工作

  1. wepe微PE工具箱 - 下载 (wepe.com.cn) 根据要刷的固件选择64位还是32位
  2. openwrt的img文件Bleach OpenWrt | lede/x86_64/2022-04-08__06-22-01–self-daily (mpdn.fun)或者【2022-04-08更新】5.15.32内核x86_64位OpenWrt固件 丰富实用插件加持 6个可选版本 - 软路由,x86系统,openwrt(x86),Router OS 等 - 恩山无线论坛 - Powered by Discuz! (right.com.cn)
  3. physdiskwrite工具https://m0n0.ch/wall/physdiskwrite.php

第一步：刷PE系统

  1. 插入u盘
  2. 打开wepe
  3. 其他安装方式选择u盘安装
  4. 选择u盘安装即可

注意：如果下载的是64位的镜像则要下载对应的64位pe系统，不然就会安装出错！

第二步：将相关文件放入u盘中

包括

openwrt的镜像

我这里是使用的恩山论坛大佬们编译的固件。如果有其他需求的朋友可以去论坛找其他固件，当然也可以自己编译啦。

不过，镜像选择最主要的功能就是看支不支持ipv6，因为其他的大部分插件是可以后期再opkg安装上去的，而ipv6的功能必须在固件编译之前就要集成好。

下载好的镜像，解压后，大概有几百M

建议改名为openwrt.img方便后续调用

physdiskwrite

这个直接在官网下好，放在u盘即可

第三步：软路由上机

  1. 将u盘插入软路由中
  2. 开机后狂按delete，或者其他键，进入bios
  3. 在boot界面设置启动项，将u盘设置第一位
  4. 重启后，进入pe系统即算成功

第四步：删除分区（可选）

新硬盘可跳过这一步

  1. 进入wepe后，选择分区工具
  2. 删除全部分区
  3. 新建分区

第五步：刷入openwrt

  1. win+R调用运行
  2. 输入cmd，调出cmd界面
  3. 输入u: 进入u盘的根目录下（有可能磁盘名字不一样，建议进入此电脑查看u盘的名称是什么）
  4. 输入physdiskwrite -u openwrt.img（这里img文件的名字就是那个openwrt镜像的名字）
  5. 选择要写入的硬盘，一般来说选择第一个 0（反正不要写到u盘里就行）
  6. 输入y
  7. 等待写入完成

第六步：检查安装情况

  1. 重启pe系统，然后迅速拔出u盘
  2. 正常情况，则会看到openwrt的加载界面了，一直跳代码，看起来还挺高级
  3. 等代码不动了，按回车，即可看到openwrt的欢迎界面
  4. 成功

第七步：管理界面检查

  1. 用一根网线将电脑与软路由的lan口连接，注意不要连接到wan口了
  2. 在浏览器中输入192.168.1.1进入管理界面
  3. 一般默认的密码是password（根据下载的固件，编译者可能有不同想法）
  4. 进入到openwrt管理界面，即大功告成
