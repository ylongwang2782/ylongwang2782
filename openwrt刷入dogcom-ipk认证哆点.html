openwrt刷入dogcom.ipk认证哆点

发表于 2022-04-19 分类于 校园网折腾 Waline： 阅读次数： 本文字数： 1.4k 阅读时长 ≈ 1 分钟

登录信息抓包

需要的软件为

1.wireshark

  1. 将网线连接电脑
  2. 打开客户端，输入账号和密码
  3. 打开wireshark，点击以太网，点击start
  4. 登录哆点
  5. 大概十几秒后，点击左上角红色按钮停止
  6. File-Save As 保存为dr.pcapng

2.生成配置文件

  1. 登录在线配置生成器https://drcoms.github.io/drcom-generic/
  2. 选择对应的哆点版本
  3. 将之前获得的dr.pcapng上传
  4. 获得新的配置文件，点击SAVE保存配置文件drcom.conf
  5. 用记事本打开配置文件drcom.conf，再对应的password处填入密码并保存

openwrt安装dogcom插件

将电脑连接到路由器的lan口，路由器的wan口连接到学校光猫

重启网卡，为了重新分配ip地址

  1. 打开winscp软件，文件协议选择SCP，地址输入路由器管理地址，账号，密码
  2. 将dogcom.ipk and drcom.conf两个文件上传到/usr/目录下
  3. 打开putty，输入地址，账号密码，登录路由器
  4. 输入cd /usr/
  5. 输入opkg install dogcom.ipk

配置dogcom

dogcom插件安装完成后，还没有用户的相关信息，所以还不能上网

  1. 重新打开putty，登录路由器
  2. 输入dogcom -m dhcp -c /usr/drcom.conf
  3. 打开浏览器检查是否能够正常上网

设置dogcom开机自启

  1. 登录winscp
  2. 打开路由器下/etc/rc.local文件
  3. 在exit 0 前加上dogcom -m dhcp -c /usr/drcom.conf
  4. 这样路由器每次启动都会自动运行该命令，实现开机自动认证哆点
